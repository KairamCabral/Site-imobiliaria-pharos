## Especificações Técnicas

- **Formato**: WebP (otimizado para web)
- **Qualidade**: Alta (90%)
- **Dimensões Recomendadas**: 
  - Capa: 1200x800px (proporção 3:2)
  - Galeria: 1200x800px (proporção 3:2)
  - Destaque: 1200x800px (proporção 3:2)
- **Peso**: Máximo 300KB por imagem
- **Orientação**: Paisagem (horizontal)

## Como Adicionar Imagens

1. Coloque as imagens nesta pasta seguindo os nomes listados acima
2. Certifique-se de que estão no formato .webp
3. Otimize as imagens antes de adicionar (pode usar ferramentas online como Squoosh)

## Placeholders

Enquanto as imagens reais não forem adicionadas, o sistema usará imagens de placeholder do Unsplash automaticamente como fallback.
